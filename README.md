🃏 Yu-Gi-Oh Card Game - Clone

Este projeto consiste em um clone inspirado no universo de Yu-Gi-Oh!, desenvolvido utilizando HTML, CSS e JavaScript.

A aplicação simula uma interface interativa de cartas, permitindo explorar elementos visuais e interações típicas de jogos de cartas digitais. O projeto foi desenvolvido com foco em prática de desenvolvimento front-end, organização de arquivos e manipulação de elementos na interface.

🚀 Tecnologias Utilizadas

HTML5 – Estrutura da aplicação

CSS3 – Estilização e layout do projeto

JavaScript – Lógica e interatividade

Assets gráficos – Cartas, fontes e imagens do projeto

🎮 Funcionalidades

✔️ Interface inspirada no universo Yu-Gi-Oh
✔️ Layout organizado e responsivo
✔️ Interação com cartas através de JavaScript
✔️ Estrutura modular de arquivos
✔️ Separação de estilos, scripts e assets

📂 Estrutura do Projeto
projeto-clone-yu-gi-oh/
│
├── index.html
│
└── src/
    ├── css/
    │   └── style.css
    │
    ├── js/

Essa estrutura permite uma organização mais profissional do projeto, separando os diferentes tipos de arquivos utilizados.
